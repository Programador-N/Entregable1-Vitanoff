Simulador de Inventario - Negocio de Ropa

Este simulador es de un catálogo de inventario de ropa. Permite ingresar, procesar y visualizar tres tipos de prenda. El usuario puede agregar nuevas prendas al inventario (de tres tipos), para luego venderlas, y visualizar el inventario actual de prendas usando la consola de JavaScript (click derecho > inspeccionar).
